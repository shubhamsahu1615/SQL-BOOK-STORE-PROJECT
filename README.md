# SQL-BOOK-STORE-PROJECT

 Description: Designed and implemented a SQL-based database system for an Online
 Book Store using PostgreSQL (pgAdmin4). The project demonstrates efficient data
 management by importing real-world datasets from Excel and applying relational
 database principles to handle customer transactions and book inventory.
 Key Features:
 Imported and managed three main datasets (Excel files):
 Customers.xlsx: Stored customer details (Name, ID, Contact,Email,City,Country.)
 Books.xlsx: Contained book information (Title, Author, Genre, Price, Stock,Book ID)
 Orders.xlsx: Recorded customer orders and transactions(Order ID,Customer ID,Book
 ID,Order Date,Quantity,Total Amount)
